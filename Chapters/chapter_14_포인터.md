# Chapter 14 포인터

포인터는 메모리 주소를 값으로 갖는 타입이다.

## 포인터 변수 선언

포인터 변수는 가리키는 데이터 타입 앞에 `*` 을 붙여서 선언한다.

```go
var p *int // int 타입 변수를 가리키는 포인터
```

```go
// package , import 기술 생략
func main() {
    var a int = 500
    var p *int // 포인터 변수 선언

    p = &a // a의 메모리 주소를 변수 p의 값으로 대입

    fmt.Printf("p의 값: %p\n", p)
    fmt.Printf("p가 가리키는 메모리의 값: %d\n", *p)

    *p = 100 // p가 가리키는 메모리 공간의 값을 변경
    fmt.Printf("a의 값: %d\n", a)
}

// <Result>
// p의 값: 0xc00000a0d8
// p가 가리키는 메모리의 값: 500
// a의 값: 100
```

### 포인터를 사용하는 이유

메모리의 주소를 직접 참조하므로, 변수 대입이나 함수 인수 전달시 또는 큰 메모리 공간을 복사할 때 발생하는 메모리 점유 문제 및 성능 문제를 완화하기 위함.
