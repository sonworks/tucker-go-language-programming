## GO 언어 특징

| 개념 | 있다/없다 | 설명 |
| --- | --- | --- |
| 클래스 | X | 단, 메서드를 가지는 구조체를 지원한다 |
| 상속 | X | 미지원 |
| 메서드 | O | 구조체가 메서드를 가질 수 있다 |
| 인터페이스 | O | 상속은 없지만, 인터페이스는 있다 |
| 익명 함수 | O | 함수 리터럴이라는 이름으로 지원한다 |
| 가비지 컬렉터 | O | 고성능 가비지 컬렉터를 지원한다 |
| 포인터 | O | 메모리 주소를 가리키는 포인터가 있다 |
| 제네릭 프로그래밍 | X | 미지원
| 네임스페이스 | X | 단, 모든 코드는 패키지 단위로 분리된다 |
